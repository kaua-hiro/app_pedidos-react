# 🍔 DevFood - Simulador de Pedidos Profissional

O **DevFood** é uma aplicação de simulação de pedidos desenvolvida com **React Native**, focada em entregar uma experiência de usuário (UX) moderna e fluida. O projeto foi estruturado seguindo as melhores práticas de desenvolvimento, com separação de responsabilidades e regras de negócio isoladas.

---

## 🚀 Demonstração

A aplicação está disponível online através do GitHub Pages:  
👉 [**Acesse o DevFood Web**](https://kaua-hiro.github.io/app_pedidos-react/)

---

## ✨ Funcionalidades

- **Escolha de Itens:** Seleção intuitiva de lanches e bebidas através de componentes Picker personalizados.
- **Visualização em Tempo Real:** Imagens dinâmicas e resumo do pedido atualizados instantaneamente.
- **Cálculo Automático:** Regras de negócio integradas para soma de valores e formatação de moeda (BRL).
- **Design Responsivo:** Interface otimizada para dispositivos móveis com limite de largura para visualização em desktops.
- **Confirmação de Pedido:** Resumo detalhado via alerta nativo e interface de "cupom fiscal".

---

## 🛠️ Tecnologias e Ferramentas

- **React Native / Expo:** Desenvolvimento multiplataforma (iOS, Android e Web).
- **JavaScript (ES6+):** Lógica avançada e manipulação de estados.
- **Regras de Negócio Isoladas:** Utilização de utilitários para cálculos e formatação.
- **GitHub Pages:** Hospedagem automatizada da versão web.
- **Arquitetura Modular:** Organização por pastas (components, constants, screens, utils).

---

## 🏗️ Estrutura do Projeto

    src/
     ┣ components/  # Componentes reutilizáveis de interface
     ┣ constants/   # Dados estáticos (cardápio, preços, imagens)
     ┣ screens/     # Telas principais da aplicação
     ┗ utils/       # Lógica e regras de negócio

## 💻 Como rodar o projeto localmente

1. **Clonar o repositório:**
    git clone https://github.com/kaua-hiro/app_pedidos-react.git

2. **Entrar na pasta do projeto:**
    cd app_pedidos-react

3. **Instalar as dependências:**
    npm install

4. **Iniciar o servidor do Expo:**
    npx expo start

5. **Acessar via Web:**
    Pressionar "w" no terminal para abrir no navegador.

---

## 📈 Diferenciais do Projeto

Diferente de um app básico, este projeto implementa:
- **Shadow/Elevation:** Profundidade visual profissional em cards.
- **Intl API:** Formatação de moeda nativa brasileira de forma robusta.
- **Clean Code:** Código modular, sem comentários desnecessários e fácil de manter.
- **UI Mobile-First:** Container com maxWidth para garantir que a experiência web simule fielmente um smartphone.
